How it works

Transparent by design

  1. A domain lookup first checks the local tenant database for a known relationship.
  2. If allowed, the service can resolve the tenant directly from Microsoft identity metadata.
  3. When more sibling domains are available, upstream enrichment expands the tenant profile.

Privacy

Clear operating mode

Privacy mode status is loaded live. When privacy mode is enabled, searches stay local and external calls are blocked.

  • No account or session is required to search.
  • Local-only mode can be enforced at the server level.
  • Every result shows what source paths were used.
